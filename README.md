# Deki LovyanGFX Integration

LovyanGFX display and touch panel driver integration for the Deki Engine.

Part of the [Deki Engine](https://github.com/ATeam-Dekiru/deki-engine) module ecosystem.

## Installation

In the Deki Editor Package Manager, add this repository URL.

## Dependencies

| Dependency | Type |
|---|---|
| `LovyanGFX` (1.2.19) | External (FreeBSD License) |
